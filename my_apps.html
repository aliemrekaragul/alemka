My Apps

  • My AES Engine

    An automated essay grading application. It uses a machine learning model trained with the prominent ASAP dataset.

  • Funny Statistics Bot

    A custom chatbot designed to make jokes about statistics while chatting. It is based on Meta's open-source LLM - llama3.1:70B.

  • Test Analyze-R

    A powerful tool I developed for TOBB ETU DFL for analyzing test results, providing comprehensive statistical insights and visualizations to help item writers make data-driven decisions. What makes it special is that it is customized to work directly with the output from the department's OCR machines.

  • ITP Score Calculator

    A practical score predictor for TOBB ETU DFL students' final language proficiency examination. Of course, everybody who is interested in their TOEFL ITP scores can use it.

  • Eksi-kazaR

    An automated web scrapping tool customized for Ekşi Sözlük, a participatory dictionary containing the comments of registered authors similar to Reddit. The tool creates word clouds in HTML format.
